# Phu luc - Goi y bo sung

Tai lieu nay dung de ghi chu cac phu luc nen chen vao ban bao cao hoan chinh.

## Phu luc A. Toan bo schema SQL

Nen chen toan bo noi dung file:

- `backend/lms.sql`

Neu phu luc qua dai, co the trich phan schema chinh va de file day du o cuoi bao cao.

## Phu luc B. Data Dictionary chi tiet

Nen lap bang cho tung bang theo cot:

- Ten bang
- Ten cot
- Kieu du lieu
- Khoa chinh
- Khoa ngoai
- Rang buoc
- Y nghia

## Phu luc C. Truy van mau

Nen dua:

- truy van thong ke hoc vien theo khoa hoc
- truy van tien do hoc tap
- truy van ket qua quiz
- truy van kiem tra khoa hoc public

## Phu luc D. Minh chung toi uu

Nen chen:

- `SHOW INDEX`
- `EXPLAIN`
- anh chup ket qua truy van

## Phu luc E. Minh chung sao luu va phuc hoi

Nen chen:

- lenh backup
- lenh restore
- anh file backup
- anh ket qua restore thanh cong
